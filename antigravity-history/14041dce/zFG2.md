# Mobile Porting Comparison

| Option | Tech | Time Est. | Compatibility | Best For |
|--------|------|-----------|---------------|----------|
| **PWA** | Web Manifest | 1-2 hrs | Browser-based | Fast validation, no store requirements. |
| **Capacitor** | Web View Wrapper | 1-2 days | Store (App/Play) | Reusing 100% of existing web code with native access. |
| **React Native** | Native UI Components | 2-4 weeks | Store (App/Play) | High performance, native feel. |

## Costs (as of 2026)
- **PWA**: Free.
- **Google Play Store**: $25 USD (one-time).
- **Apple App Store**: $99 USD (annually).

## Distribution
- **Android**: Can be distributed via APK download link (Sideloading) for free.
- **iOS**: Strictly requires Apple Developer Program/TestFlight for most distribution.
