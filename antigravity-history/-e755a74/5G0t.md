# Delete Booking Feature

## Goal
Allow Admin to permanently delete a booking from the Edit Modal.

## Changes

### Backend
**File:** `src/app/api/admin/bookings/[id]/route.ts`
- Add `DELETE` method.
- Check Admin role.
- Execute `delete()` on `bookings` table.

### Frontend
**File:** `src/components/admin/BookingEditModal.tsx`
- Add "Eliminar" button (red, with trash icon) next to "Cancelar".
- Implement `handleDelete` function:
    - `window.confirm('¿Estás seguro de que quieres eliminar esta reserva? Esta acción no se puede deshacer.')`
    - Call `DELETE /api/admin/bookings/${booking.id}`.
    - On success: `onSave()` (which triggers refresh) and close modal.

## Verification
- Create a test booking.
- Open it.
- Click "Eliminar".
- Confirm.
- Verify it disappears from the grid.
